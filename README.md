# Sufa Logistik Company Profile

Static company profile website for Sufa Logistik.

## GitHub Pages

This project is ready to deploy with GitHub Pages using GitHub Actions.

1. Push this folder to a GitHub repository.
2. Open the repository settings.
3. Go to `Pages`.
4. Set `Build and deployment` source to `GitHub Actions`.
5. Push to `main` or `master`, or run the `Deploy GitHub Pages` workflow manually.

The demo URL will appear in the workflow summary after deployment.
